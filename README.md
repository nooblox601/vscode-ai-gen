# vscode-ai-codegen

## Overview

vscode-ai-codegen is a Visual Studio Code extension that leverages artificial intelligence to generate code snippets and frameworks based on user input. This extension aims to enhance developer productivity by providing intelligent code generation capabilities directly within the editor.

## Features

- **Code Generation**: Generate code snippets in various programming languages based on user-defined parameters.
- **Framework Creation**: Create boilerplate code for popular frameworks with ease.
- **User-Friendly Interface**: Interactive webview panel for seamless user experience.
- **AI-Powered**: Utilizes advanced AI models to provide accurate and relevant code suggestions.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/vscode-ai-codegen.git
   ```
2. Navigate to the project directory:
   ```
   cd vscode-ai-codegen
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Open the project in Visual Studio Code:
   ```
   code .
   ```

## Usage

1. Open the command palette (Ctrl+Shift+P or Cmd+Shift+P).
2. Type `Generate Code` to invoke the code generation command.
3. Follow the prompts to specify the desired code snippet.
4. Type `Generate Framework` to create a new framework setup.
5. Choose from the available framework options and follow the instructions.
